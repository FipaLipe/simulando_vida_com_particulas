# Como executar?

Python 10+ é necessário para rodar o código. o procedimento recomendado é criar um novo ambiente virtual e instalar os módulos necessários com

```sh
pip install -r requirements.txt
```

Para rodar a versão final do projeto, execute o arquivo `main.py` com

```sh
python codigos/main.py
```

Para entender a base teórica do projeto, leia o arquivo `simulando_vida_com_particulas.ipynb` na pasta _codigos/_.

---

# Sobre o Repositório

Este repositório tem como objetivo visualizar a aparição de `fenômenos emergentes` em sistemas complexos com regras iniciais relativamente simples.

Para isso, foi desenvolvido uma simulação de partículas distintas com interações definidas, inspirado no trabalho de [Clusters de Jeffrey Ventrella](https://www.ventrella.com/Clusters/intro.html). A partir dessa simulação, observa-se a emergência de uma suposta complexidade, com estruturas que se assemelham àquelas observadas na natureza biológica.
